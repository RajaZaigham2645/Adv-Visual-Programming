# 🚗 Car Showroom Management System C#

A **desktop-based Car Showroom Management System** developed using **C# and Windows Forms**.
This application helps manage car inventory, customers, and sales operations in a simple and efficient way.

---

## 📌 Project Overview

This system is designed for showroom staff to easily:

* Manage car records
* Handle customer information
* Process sales transactions
* Keep track of available and sold vehicles

It provides a user-friendly interface built with **Windows Forms GUI**.

---

## 🛠️ Technologies Used

* **Language:** C#
* **Framework:** .NET (Windows Forms)
* **IDE:** Visual Studio Code / Visual Studio
* **UI:** Windows Forms (WinForms)
* **Database:** SQL Server / Local Database (if used)

---

## ✨ Features

### 🚘 Car Management

* Add new car details (Model, Brand, Price, Status)
* Update existing car records
* Delete car information
* View all available cars

### 👤 Customer Management

* Add customer details
* Update customer information
* Store customer purchase records

### 💰 Sales Management

* Record car sales
* Link customers with purchased cars
* Track sold vehicles

### 🖥️ User Interface

* Simple and clean Windows Forms UI
* Easy navigation between forms
* Interactive buttons and controls
